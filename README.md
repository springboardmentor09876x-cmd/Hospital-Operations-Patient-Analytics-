# MedTrack_DV - Hospital Operations & Patient Analytics Dashboard

## Module 1: Hospital Data Collection

### Project Overview
MedTrack_DV is a Hospital Operations and Patient Analytics project designed to analyze hospital performance, patient admissions, department efficiency, and healthcare resource utilization. This project will eventually include interactive Tableau dashboards for healthcare analytics.

This repository currently contains the deliverables for **Module 1**, which focuses on collecting and validating hospital operational data.

---

## Module 1 Objectives

- Collect hospital operational data
- Prepare a raw dataset for analysis
- Verify dataset completeness
- Perform basic dataset validation
- Maintain data integrity for future processing

## Dataset

**File:** `hospital_raw_data.csv`

The dataset contains hospital operational records, patient information, department details, admission records, treatment information, and healthcare resource data.

---

## Python Script

**File:** `data_collection.py`

The script performs the following tasks:

- Loads the hospital dataset
- Verifies dataset availability
- Displays dataset summary
- Prints column names
- Displays data types
- Shows sample records
- Checks missing values
- Checks duplicate records

No data cleaning or transformation is performed in Module 1.
