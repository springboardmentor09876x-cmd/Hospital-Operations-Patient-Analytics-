MedTrack\_DV: Hospital Operations \& Patient Analytics Dashboard



 📊 Project Statement

The objective of this project is to develop a comprehensive hospital operations and patient analytics dashboard suite for monitoring hospital performance, patient admissions, department efficiency, and resource utilization using interactive Tableau dashboards.




🚀 Repository \& Workflow Progress



 🟢 Milestone 1: Data Collection \& Preparation (Weeks 1-2)



Module 1: Hospital Data Collection

\* \*\*Tasks completed:\*\* Downloaded operational datasets, collected patient admission logs, integrated secondary resource schemas.

\* \*\*Deliverables:\*\* `data/hospital\_raw\_data.csv`, `src/data\_collection.py`

\* \*\*Evaluation:\*\* Achieved 100% dataset integration, maintaining data completeness well above the 95% threshold requirement.



\#### Module 2: Data Cleaning \& Transformation

\* \*\*Tasks completed:\*\* Removed duplicate entries, handled structural missing values in medication and equipment records, standardized department names, and normalized healthcare indicators into a Tableau-ready output.

\* \*\*Deliverables:\*\* `data/hospital\_cleaned.csv`, `notebooks/hospital\_cleaning.ipynb`

\* \*\*Evaluation:\*\* Maintained 0% missing values across all critical metrics (exceeding the requirement of <2% missing values).

\## 📁 Project Layout

\* \*\*`data/`\*\*: Holds raw data records and the validated, Tableau-ready clean data.

\* \*\*`src/`\*\*: Executable python script verifying data integrity.

\* \*\*`notebooks/`\*\*: Interactive step-by-step documentation of the data cleaning pipeline.
